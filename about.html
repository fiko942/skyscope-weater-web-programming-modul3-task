Tentang Aplikasi

SkyScope Weather adalah aplikasi prakiraan cuaca multipage sederhana yang memanfaatkan Fetch API dan async/await untuk menghadirkan data real-time tanpa proses build berat. Semua antarmuka dibangun dengan Tailwind CSS CDN dan interaksi UI memakai jQuery.

Sumber Data Utama

Nominatim (OpenStreetMap)

Ketika pengguna mencari kota, aplikasi mengirim permintaan GET ke Nominatim pada endpoint https://nominatim.openstreetmap.org/search. Respons JSON menyediakan koordinat latitude dan longitude serta display_name untuk ditampilkan di antarmuka. API ini tidak membutuhkan kunci dan ramah untuk eksperimen pendidikan.

Open-Meteo Forecast API

Dengan koordinat yang sudah diketahui, aplikasi melakukan permintaan GET kedua ke Open-Meteo https://api.open-meteo.com/v1/forecast. Parameter yang digunakan mengaktifkan current_weather, data harian (7 hari), serta data per jam (minimal 12 jam). Open-Meteo dipilih karena gratis, tanpa API key, dan dokumentasinya jelas.

Alur Fetch dengan Async/Await

  1. 1. Ambil input pengguna atau lokasi saat ini — pencarian kota dari form atau koordinat geolokasi.
  2. 2. await fetch() ke Nominatim — hasil dikonversi ke JSON setelah memeriksa response.ok.
  3. 3. await fetch() ke Open-Meteo — parameter disusun dinamis, JSON diambil dan divalidasi.
  4. 4. Transformasi data — memetakan weathercode, memformat angka, dan menyusun struktur kartu.
  5. 5. Render UI dengan jQuery — skeleton disembunyikan, konten ditampilkan, dan toast akan muncul untuk status.
  6. 6. Cache di localStorage — data terakhir disimpan selama 10 menit untuk pengalaman semi-offline.

Mengapa Kombinasi Teknik Ini?

Tailwind CDN mempercepat prototyping tanpa konfigurasi build. jQuery masih relevan untuk manipulasi DOM sederhana, animasi ringan, dan aksesibilitas event di seluruh browser. Fetch API dengan async/await menghasilkan alur kode yang mudah diikuti dan memaksimalkan kemampuan JavaScript modern.

Aplikasi ini menjaga aksesibilitas lewat struktur semantik, fokus ring, warna kontras, dan skeleton dengan role="status". Untuk SEO ringan, halaman memiliki meta description, judul deskriptif, serta tautan internal ke halaman About dan Home.
